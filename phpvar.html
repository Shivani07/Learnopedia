PHP variables

In PHP, a variable starts with the $ sign, followed by the name of the variable

Example

< ?php
$x=5;
? >

The above statements will decalre a variable 'x' with the value '5' in it.
Unlike other programming languages, PHP has no command for declaring a variable. It is created the moment you first assign a value to it.
NOTE:PHP is case sensitive i.e. $x and $X are treated differently.

Rules for PHP variables
  • A variable starts with the $ sign, followed by the name of the variable.
  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive ($age and $AGE are two different variables)

Variable Scope

In PHP, variables can be declared anywhere in the script.
The scope of a variable is the part of the script where the variable can be referenced.
PHP has three different variable scopes:
Global
A variable declared outside a function has a GLOBAL SCOPE and can only be accessed outside a function.

example

< ?php
$x=5; //global variable
function
myFunc(){
echo
"value of variable is ".$x;
}
? >

Output

value of variable is 5
The Global keyword is used to declare the variable global.

Example

function
myFunc(){
global $x=5; echo
"value of variable is ".$x;
}
Local
A variable declared within a function has a LOCAL SCOPE and can only be accessed within that function

example

< ?php
function
myFunc(){
$x=5; //local variable
echo
"value of variable is ".$x;
}
? >

Output

value of variable is 5
Static
A static variable is not deleted even after the execution of the function.
It is once initialized does not loose its value between different function calls.
We use static keyword to declare a variable static.

Example

< ?php
function
myFunc(){
static $x=5; //static variable
echo
"value of variable is ".$x;
$x=$x+2; }
? >
myFunc();
myFunc();
myFunc();

Output

value of variable is 5
value of variable is 7
value of variable is 9
